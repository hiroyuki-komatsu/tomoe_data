# tomoe_data: Japanese Handwriting Stroke Data.

Tomoe is a handwriting recognition tool, I ([Hiroyuki Komatsu](https://github.com/hiroyuki-komatsu/)) originally created and later handed over to other project contributors.

Official Project: https://sourceforge.net/projects/tomoe/

This repository (https://github.com/hiroyuki-komatsu/tomoe_data/) contains the stroke data that was part of my own contribution to the project.
